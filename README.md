# Deep-Learning-Framework-for-Multiple-Sclerosis-Lesion-Segmentation-and-Filling-in-Brain-MRI
This project develops a deep learning model to perform lesion filling in brain MRI scans. Using masked FLAIR images as input and original FLAIR images as targets, a U-Net network learns to reconstruct missing brain tissue and generate lesion-filled MRI slices for improved brain structure analysis.
